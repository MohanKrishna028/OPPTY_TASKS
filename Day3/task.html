Headings & Paragraphs

Learn how to create headings and paragraphs in HTML

HTML Headings

HTML headings are defined with the <h1 > to <h6> tags.
<h1> is the most important heading, and< h6 >is the least important.

Important Heading Rules

  • Use only one <h1> per page - this should be the main title
  • Don't skip heading levels (don't jump from<h2> to <h4>)
  • Headings should be used to structure content hierarchically
  • Headings are important for SEO and accessibility

Unordered Lists
