Links and Anchors

Learn how to create hyperlinks in HTML

HTML Links

Links are created using the <a> (anchor) element. The href attribute specifies the destination of the link.

Types of Links

Absolute URLs

Links to external websites with full URLs:
Example:- this is a external url Google

Relative URLs

Links to pages within the same website:
Example:- back to top
