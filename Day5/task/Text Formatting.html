HTML Text Formatting Elements

HTML provides several elements to format text and make it more visually appealing or semantically meaningful.

Basic Text Formatting

Bold Text

Use <b> for bold text without semantic importance or <strong> for text of great importance:
Example:- this text is bolded by using <b> tag

Italic Text

Use <i> for italic text without semantic importance or <em> for emphasized text:
Example:-This entir senternse is makes to italis using the <i> tag

Underlined and Marked Text

Use <u> for underlined text or <mark> to highlight text:
Example:- This ebtire sentense is underlined by using <u> tag  and this sentense is mrked by using <mark>

Example:- This Entire Sentense size is decresed by using <small> tag

Quotations

Use <q> for short quotations
Example:- This setes is places inside the quots using <q>
