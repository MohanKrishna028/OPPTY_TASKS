Lists

Learn how to create different types of lists in HTML

HTML Lists

HTML provides three types of lists: ordered lists, unordered lists, and definition lists.

Unordered Lists

Unordered lists are created with the <ul> tag, and list items with <li>:
Example
  • List1
  • list2
  • list3

Ordered List

Ordered lists are created with the <ol> tag, and list items with <li>:
Example
  1. List1
  2. list2
  3. list3
  1. List1
  2. list2
  3. list3
  1. List1
  2. list2
  3. list3

Definition Lists

Definition lists are created with the <dl> tag, terms with <dt>, and descriptions with <dd>:
Mango
It is fruit
Tomato
It is a vegitale
car
It is road vehicle
ship
It is water vhicle

It is nothig but creating list inside list
Example
